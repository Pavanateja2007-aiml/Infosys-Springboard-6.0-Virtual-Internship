# IRCTC IVR System

FastAPI based Railway IVR system using Twilio voice API.

Features:
- Language selection
- PNR enquiry
- Train running status
- Train schedule

Tech Stack
- FastAPI
- Twilio Voice API
- Python
- Ngrok
